# Ecommerce microservice application

## How to setup

Requirement:
- Nodejs 22 LTS
- Java 21 LTS

For **Nodejs** packages, only need to run `yarn` at the root folder.

## How to start

To start the project in development mode, you can use this commands on root folder:

```bash
yarn dev
```
